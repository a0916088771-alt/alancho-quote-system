# alancho_0 攝影報價單系統

韓系、乾淨、高留白的網頁版攝影報價單產生器。支援客戶資料、服務項目與加購項目計算、訂金與尾款、自訂 Logo、付款 QR Code、手寫電子簽名、歷史報價 localStorage 儲存，以及一鍵列印匯出 PDF。

## 安裝

```bash
npm install
```

## 啟動

```bash
npm run dev
```

打開瀏覽器前往：

```text
http://localhost:3000
```

## 匯出 PDF

在頁面右上角點選「匯出 PDF」，系統會開啟瀏覽器列印視窗。目的地選擇「儲存為 PDF」即可輸出正式報價單，內容會包含品牌、客戶資料、明細、金額、付款資訊、注意事項與簽名。

## 資料儲存

所有報價資料、Logo、QR Code 與簽名會儲存在目前瀏覽器的 `localStorage`，不會上傳到伺服器。若更換瀏覽器或清除網站資料，歷史報價會消失。
